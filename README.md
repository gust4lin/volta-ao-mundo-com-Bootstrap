## 📌 Sobre o Projeto
O *Volta ao Mundo — Colômbia* é uma aplicação web que tem como objetivo apresentar a Colômbia por meio de uma experiência digital organizada e intuitiva.

A proposta do projeto é transformar informações culturais, históricas e naturais em uma navegação fluida, permitindo que o usuário explore o país como em uma pequena viagem virtual pelos Andes, pelo Caribe e pela Amazônia colombiana.

A estrutura do conteúdo foi planejada para equilibrar *informação, estética e leveza*, reunindo desde o café e as esmeraldas até o ritmo da salsa, o carnaval e as praias paradisíacas.
